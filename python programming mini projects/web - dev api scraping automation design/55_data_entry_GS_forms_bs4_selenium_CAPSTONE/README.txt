Scrape fake Zillow listings and auto-submit their data to a Google Form.

Key Steps:
Use BeautifulSoup to parse listings page.
Extract addresses, prices, and links from property cards.
Open Google Form in Selenium.
Fill out address, price, and link fields.
Submit the form and optionally continue to next form.
Can loop through multiple listings automatically.

Notes:
Handles Google Form input fields via XPath and CSS selectors.
Can integrate with live Zillow scraping by changing the URL.
