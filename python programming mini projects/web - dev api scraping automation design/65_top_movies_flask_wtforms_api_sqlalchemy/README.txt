Flask web application for managing a personal movie collection. Users can:

View all movies
Add movies using the TMDb API
Edit movie ratings and reviews
Delete movies

The app uses:
Flask for the web framework
Flask-WTF for forms and validation
SQLAlchemy for ORM and SQLite database
TMDb API to search for movie details
Flask-Bootstrap 5 for styling

All dependencies are listed in requirements_3.13.txt.
Must activate venv.
