1. Birthday Wisher (Python – SMTP + Pandas + dotenv)

Automatically sends personalized birthday emails using Gmail.
Reads birthdays from a CSV file, selects today’s matches, and fills a random letter template.
Uses smtplib for email, pandas for CSV processing, and dotenv for secure credentials.
If no birthdays today, script does nothing.

2. Weekly Motivational Email Sender (Python – SMTP + dotenv)

Sends a motivational quote via email every Thursday.
Reads quotes from a text file, randomly selects one, and formats the message.
Uses smtplib to send emails and dotenv to safely store credentials.
Automates a small newsletter directly from Python.
