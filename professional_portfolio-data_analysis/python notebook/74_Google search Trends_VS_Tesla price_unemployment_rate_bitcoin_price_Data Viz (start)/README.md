Project Overview

This project investigates whether Google search popularity can be linked to real-world economic and financial indicators. By combining Google Trends data with external datasets, the notebook explores patterns and correlations between public interest and measurable outcomes.

Questions Explored
Does Google search interest correlate with unemployment rates?
Can search trends help explain movements in Bitcoin prices?
Is there a relationship between Google searches and Tesla stock prices?

Data Sources
Google Trends – Search interest over time
FRED – U.S. unemployment rate
Yahoo Finance – Historical data for Tesla and Bitcoin

Key Techniques
Time-series data alignment
Data normalization
Correlation analysis
Advanced data visualization
