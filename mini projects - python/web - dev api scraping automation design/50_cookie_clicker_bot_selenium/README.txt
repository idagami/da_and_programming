Automate clicking in the Cookie Clicker game and buy upgrades automatically.

Key Steps:
Open the game page with Selenium Chrome WebDriver.
Click the “English” language button.
Continuously click the big cookie in a loop for 3 minutes.
Periodically check for unlocked upgrades and buy the most expensive affordable one.
Print cookies per second at the end.

Notes:
Handles stale elements and exceptions gracefully.
Uses time intervals to check upgrades and prevent CPU overuse.
