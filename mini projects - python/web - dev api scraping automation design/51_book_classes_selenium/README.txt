Log in to a web-based gym app and automatically book Tuesday & Thursday classes at a target time.

Key Steps:
Configure Selenium Chrome with a custom profile.
Login using email and password.
Search for desired class time in Tue/Thu sections.
Click “Book class” or “Join waitlist” buttons.
Retry booking if some classes are missing.
Print a summary of booked/waitlisted classes.

Notes:
Includes robust retry mechanisms for failures.
Checks actual bookings vs available slots.
