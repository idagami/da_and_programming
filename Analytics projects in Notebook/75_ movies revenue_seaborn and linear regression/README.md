Project Overview

This project analyzes whether higher movie budgets lead to higher box office revenue. Using scraped data from the-numbers.com, the notebook applies data visualization and linear regression to explore the relationship between production cost and financial success.

Dataset
Movie budget and box office revenue data scraped from the-numbers.com (May 1, 2018)

Questions Explored
Do movies with higher production budgets tend to generate higher box office revenue?
How strong is the correlation between budget and revenue?
Does the relationship change when using a logarithmic scale?
Are there noticeable outliers where low-budget films perform exceptionally well or high-budget films underperform?
How reliable is linear regression for predicting revenue based on budget alone?

Key Techniques
Data cleaning and filtering
Log-scale visualizations
Linear regression modeling
Correlation analysis
