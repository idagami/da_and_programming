Project Overview

This project explores the history and evolution of LEGO using real-world data from Rebrickable. The goal is to answer interesting business and product questions about LEGO sets, themes, and complexity over time using data analysis techniques in Python.

Questions Explored
What is the largest LEGO set ever created in terms of number of parts?
When did LEGO first release sets, and how did sales look in the early years?
Which LEGO themes have the most sets?
When did LEGO significantly expand its product offerings?
Have LEGO sets become more complex over time?
