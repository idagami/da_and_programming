This project focuses on analyzing the business performance of a large distributor 
operating in the US and worldwide. The goal was to create a multi-page Power BI report 
that provides actionable insights into revenue, profit, discounts, and other key metrics, 
helping stakeholders understand current challenges and identify opportunities to improve profitability.

The report is designed for a wide audience, including business managers, analysts, and decision-makers, 
allowing them to explore data interactively without technical knowledge of SQL.

Dataset: The analysis is based on the provided superstore_dataset.xlsx file

Power BI Report Structure
1. Financial Overview
2. Monthly Metrics Analysis
3. Discount Analysis
4. Additional Attribute Analysis
5. Orders Detail Page

Key Insights
Identification of top-selling products and profitable regions.
Discounts impact profit differently across categories and regions.
Certain product categories or shipping types may require further optimization.
Trend analysis supports strategic decisions for global operations.
