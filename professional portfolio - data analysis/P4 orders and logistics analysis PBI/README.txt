This project analyzes orders, employee performance, and interactions with logistics partners for a global gourmet retailer. 
The goal was to create a multi-page Power BI report that provides actionable insights into sales, discounts, 
order fulfillment, and employee performance, helping stakeholders optimize business operations and profitability.

Power BI Report Structure
1. Financial Overview
2. Logistics Partners Analysis
3. Employee Performance Analysis
4. Orders Detail Page
